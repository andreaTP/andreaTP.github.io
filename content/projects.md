A few open-source projects I significantly contributed to:

- [**Chicory**](https://chicory.dev) - A pure-Java, zero-native-dependency WebAssembly runtime for the JVM (lead maintainer)
- [**Fabric8 Kubernetes Client**](https://github.com/fabric8io/kubernetes-client) - Java client for Kubernetes, including the CRD-to-Java type generator
- [**Keycloak**](https://www.keycloak.org) - Open source identity and access management, Kubernetes Operator
- [**Kiota**](https://github.com/microsoft/kiota) - OpenAPI-based HTTP client code generator
- [**Cloudflow**](https://cloudflow.io) - Streaming data pipelines on Kubernetes
- [**Akka.Js**](https://akka-js.org) - Port of the Akka actor framework to Scala.js
